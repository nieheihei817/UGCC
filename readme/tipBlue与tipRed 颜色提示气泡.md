# tipBlue与tipRed 颜色提示气泡



## tipBlue调用示例

```vue
<tipBlue>
  <h2 id="solve">解决方法</h2>
    <li>在规定时间内完成CEO的要求</li>
    <li>在游戏开始前取消勾选“失败条件”</li>
    <li>避免大规模暴动、瘟疫和负债过多</li>
    <li>在官僚科技树中研究法律辩护(一次性)</li>
</tipBlue>
```



组件接收一个h2标签作为其标题，多个li作为子项

## tipRed调用示例

```vue
<tip-red>
    <h2>触发条件</h2>
    <li>监狱内伤亡人数过多</li>
    <li>负债过高</li>
</tip-red>
```
